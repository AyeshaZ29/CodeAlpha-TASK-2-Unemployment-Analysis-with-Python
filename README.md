***Project Overview***

This project analyzes unemployment rate trends in India using Python. It focuses on data cleaning, exploratory data analysis (EDA), visualization, and understanding the impact of Covid-19 on employment. The goal is to extract insights that can help inform economic and social policy decisions.

This project was completed as Task 2 for the CODEALPHA Data Science Internship.

*Objectives*

Analyze unemployment rate data (percentage of unemployed people)

Clean and preprocess real-world data

Visualize unemployment trends over time

Study the impact of Covid-19 on unemployment rates

Identify seasonal and monthly patterns

Compare Rural vs Urban unemployment

Present insights useful for policymakers and researchers
